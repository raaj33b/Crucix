## 📊 What is Moldavite?

Moldavite is a **real-time OSINT intelligence aggregator** that:

✅ **Aggregates trending feeds** from Reddit, Twitter/X, GitHub, HackerNews, CVEFeed, and MetaClaw  
✅ **Visualizes entity relationships** in 3D force-directed graph with 89+ pre-loaded nodes  
✅ **Maps correlations** (CVE chains, tech adoption cascades, market impacts)  
✅ **Creates linked notes** with Obsidian-style `[[entity]]` linking  
✅ **Highlights anomalies** — critical CVEs, GPU shortages, APT activity  
✅ **Predicts signal propagation** — Reddit → Twitter → GitHub in 4-6 hours

## 🚀 Quick Start

### Installation
```bash
# Clone repository
git clone https://github.com/raaj33b/crucix.git
cd crucix

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Access Points
- **Dashboard**: `http://localhost:3000/moldavite/dashboard`
- **3D Graph**: `http://localhost:3000/moldavite/graph`
- **Full Hub**: `http://localhost:3000/` (Tab 8: Moldavite)
